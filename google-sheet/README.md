# 📊 نظام تتبع العملاء — Google Sheets CRM

نظام لتنظيم كل العملاء اللي كيتواصلوا معك أو كيطلبوا مشروع عبر فورم الموقع.

> ✅ **النظام مفعّل ومربوط بالموقع منذ 2026-07-04.**
> رابط الـ Web App المنشور محفوظ في `index.html` (المتغير `SHEET_WEBAPP_URL`).
> الخطوات أدناه للمرجع فقط إذا احتجت إعادة التفعيل من الصفر.

## ⚙️ خطوات التفعيل (مرة واحدة فقط — 5 دقائق)

### 1. لصق الكود في الشيت
1. افتح الشيت ديالك: [رابط الشيت](https://docs.google.com/spreadsheets/d/1Xngxzn64JcDHlSCHPa6T-a1Dmt6GWefYGzZ2847omdc/edit)
2. من القائمة: **الإضافات (Extensions) ← Apps Script**
3. امسح أي كود موجود، والصق محتوى الملف `Code.gs` كاملاً
4. اضغط **حفظ** (أيقونة 💾)

### 2. بناء النظام
1. من القائمة العلوية فوق الكود، اختر الدالة **setupCRM**
2. اضغط **تشغيل (Run)** ▶️
3. غادي يطلب منك الإذن أول مرة: **Review permissions ← اختر حسابك ← Advanced ← Go to ... (unsafe) ← Allow**
4. ارجع للشيت — غادي تلقى 3 أوراق جاهزة:
   - **📋 الطلبات** — كل عميل في صف، مع قوائم منسدلة وألوان تلقائية حسب الحالة
   - **📊 لوحة التحكم** — إحصائيات ومداخيل محسوبة أوتوماتيكياً
   - **⚙️ الإعدادات** — لوائح الحالات والمصادر والخدمات

### 3. النشر كـ Web App (باش الفورم يرسل للشيت)
1. في Apps Script: **نشر (Deploy) ← نشر جديد (New deployment)**
2. اضغط ⚙️ بجانب "Select type" واختر **Web app**
3. الإعدادات:
   - **Execute as**: أنا (Me)
   - **Who has access**: أي شخص (**Anyone**) ← مهم جداً!
4. اضغط **Deploy** وانسخ الرابط اللي كيبدا بـ `https://script.google.com/macros/s/...`

### 4. ربط الموقع
- عطيني الرابط اللي نسختيه ونحطو في الموقع، أو بدّلو بنفسك في `index.html`:
  ```js
  const SHEET_WEBAPP_URL = 'الرابط هنا';
  ```

## ✅ كيفاش يخدم النظام

- كل ما يملأ عميل الفورم في الموقع ← يوصل تلقائياً للشيت بحالة **🆕 جديد** ومصدر **🌐 فورم الموقع**، ويفتح الواتساب عادي.
- العملاء اللي كيجيو من الواتساب مباشرة ← زيدهم يدوياً في صف جديد واختر المصدر **💬 واتساب مباشر**.
- بدّل **الحالة** من القائمة المنسدلة ولون الصف كيتبدل بوحدو (جديد → تواصل → اتفاق → تنفيذ → تسليم).
- عمّر **السعر المتفق** و**المدفوع** ← عمود **المتبقي** كيتحسب أوتوماتيكياً، والمجاميع كتبان في لوحة التحكم.

## 🔄 ملاحظات

- إلا بدلتي الكود من بعد، خاصك دير **Deploy ← Manage deployments ← ✏️ ← Version: New version ← Deploy** باش التعديل يتفعّل.
- تشغيل `setupCRM` مرة أخرى غادي يمسح بيانات ورقة الطلبات ويعيد بناءها — استعملها غير في البداية.
